# Evaluation_task_practikum

AI-Driven Learning Platform built with:

- FastAPI (Backend)
- React + TypeScript + Vite (Frontend)
- SQLite Database
- OpenAI GPT API

---

# Features

- User registration
- Categories & sub-categories
- AI-generated learning responses
- Learning history
- Admin dashboard
- REST API architecture
- TypeScript frontend
- FastAPI backend

---

# Technologies Used

## Backend

- Python
- FastAPI
- SQLAlchemy
- SQLite
- OpenAI API
- Uvicorn

## Frontend

- React
- TypeScript
- Vite
- React Router DOM
